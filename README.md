## 📋 Sobre o Projeto

Este projeto é um **Portfolio Dev** desenvolvido como um desafio prático da formação de PHP da **Rocketseat**. A aplicação consiste em uma página de portfólio pessoal moderna, responsiva e estruturada utilizando componentes PHP para melhor organização do código.

O design foi baseado na comunidade do Figma: [Portfolio Dev PHP](https://www.figma.com/community/file/1426251628808157943/portfolio-dev-php).

### ✨ Funcionalidades Principais

- **Seção Hero**: Apresentação pessoal com destaque para as tecnologias principais.
- **Destaque de Projetos**: Galeria de projetos com descrição e stacks utilizadas em cada um.
- **Seção de Contato**: Links sociais e informações de contato com design atrativo.
- **Componentização**: Uso de `require` do PHP para separar seções em arquivos distintos (`hero.php`, `projects.php`, `contacts.php`).
- **Estilização Moderna**: Utilização do Tailwind CSS (via CDN) para um desenvolvimento ágil e visual profissional.

***

## 🛠️ Tecnologias Utilizadas

- **Linguagem**: [PHP](https://www.php.net/) (Core)
- **CSS Framework**: [Tailwind CSS](https://tailwindcss.com/) (v4 via Browser CDN)
- **Tipografia**: [Google Fonts](https://fonts.google.com/) (Asap, Inconsolata, Maven Pro)
- **Ícones**: Imagens customizadas em SVG/PNG para redes sociais e projetos.

## 🚀 Como Iniciar

### 💻 Execução Local

Para visualizar o projeto localmente, você precisa ter o PHP instalado em sua máquina.

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/desafio_portfolio_dev.git
   cd desafio_portfolio_dev
   ```
2. **Inicie o servidor embutido do PHP:**
   ```bash
   php -S localhost:8000
   ```
3. **Acesse no navegador:**
   Abra o endereço `http://localhost:8000` para visualizar o portfólio.
